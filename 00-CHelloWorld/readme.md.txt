# Información del compilador

## Compilador seleccionado

Se utilizó el compilador **g++** provisto por MinGW.

## Versión del compilador

15.2.0


## Versión del estándar de C soportado

El compilador soporta múltiples estándares del lenguaje C, entre ellos:

C23 (ISO/IEC 9899:2024) 
C17 (C18) 
C11 
C99 
C89/C9
